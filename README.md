# ChatBridge — チャット翻訳ツール 🌉

ゲーム中のチャットをリアルタイムで翻訳するデスクトップツール。
ホットキー一発で入力中のテキストを翻訳し、そのまま送信できます。

## 特徴

- 🎮 **ホットキー翻訳**: `Ctrl+J` で選択/入力中のテキストを瞬時に翻訳
- 🌏 **多言語対応**: 日本語⇔英語をはじめ、9言語間の翻訳に対応
- 🔄 **オーバーレイ表示**: ゲーム画面の上に翻訳結果を表示
- ⚙️ **設定画面**: ホットキー、翻訳方向、UI言語などをカスタマイズ
- 🌐 **UIローカライゼーション**: 日本語/英語UI内蔵、外部JSONで言語追加可能
- 💰 **無料**: MyMemory翻訳API（APIキー不要）

## 使い方

### 起動

```bash
# 仮想環境の有効化
.\venv\Scripts\activate

# 起動
python main.py
```

### 翻訳操作

1. 翻訳したいテキストがあるウィンドウをアクティブにする
2. `Ctrl+J` を押してキーを離す
3. オーバーレイに翻訳結果が表示される
4. `Enter` → 翻訳結果をペースト / `Esc` → キャンセル

### 設定

システムトレイのアイコンを右クリック → 「設定を開く」

## UIローカライゼーション

日本語・英語はアプリに内蔵されています。
他の言語を追加するには:

1. アプリと同じフォルダに `locales/` フォルダを作成
2. `locales/ko.json` のようにJSONファイルを配置
3. 設定画面で新しい言語を選択

JSONファイルのフォーマットは [i18n.py](i18n.py) の `_BUILTIN` 辞書を参照してください。

## 技術スタック

- Python 3.12+
- PySide6 (Qt6) — UI
- pynput — ホットキー検出 & キー送信
- pystray — システムトレイ
- MyMemory API — 翻訳
