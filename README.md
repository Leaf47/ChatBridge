# JA2EN — ゲーム内日本語→英語翻訳ツール

ゲーム中にウィンドウを切り替えることなく、日本語を英語に翻訳できるシステムトレイ常駐型ツールです。

## 特徴

- 🎮 **ゲーム内で完結** — ウィンドウ切替不要、チャット欄でそのまま翻訳
- ⚡ **ワンキーで翻訳** — `Ctrl+J` で即座に日本語→英語変換
- 🔄 **API切り替え** — MyMemory（無料）/ DeepL / Google Cloud に対応
- 🖥️ **オーバーレイ表示** — ゲーム画面上に半透明パネルで翻訳結果を確認
- ⚙️ **カスタマイズ** — ホットキー、翻訳エンジン、表示位置を自由に設定

## セットアップ

### 1. Python のインストール

Python 3.11 以上が必要です。[python.org](https://www.python.org/) からインストールしてください。

### 2. 依存パッケージのインストール

```bash
cd e:\dev_root\ja2en_dict
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 起動

```bash
python main.py
```

システムトレイに紫色の「J」アイコンが表示されます。

## 使い方

### 基本的な使い方

1. ゲーム内のチャット欄に日本語を入力する
2. `Ctrl+J` を押す
3. オーバーレイに原文と翻訳結果が表示される
4. `Enter` で確定（英訳がペーストされる）/ `Esc` でキャンセル

### 設定の変更

- トレイアイコンを右クリック →「⚙️ 設定を開く」
- ホットキー、翻訳エンジン、オーバーレイの表示設定を変更できます

### 翻訳エンジンの切り替え

- トレイアイコンを右クリック →「🔄 翻訳エンジン」から選択
- または設定画面の「翻訳エンジン」タブで変更

## 翻訳エンジン

| エンジン | APIキー | 月間制限 | 品質 |
|---------|---------|---------|------|
| **MyMemory** | 不要 | 1日1000リクエスト | ○ |
| **DeepL Free** | 要（無料） | 50万文字 | ◎ |
| **Google Cloud** | 要（無料枠あり） | 50万文字 | ○ |

## 注意事項

- **ボーダーレスウィンドウモード**でのプレイを推奨します（排他的フルスクリーンではオーバーレイが表示されない場合があります）
- ゲームの**アンチチートソフト**との互換性は保証されません
