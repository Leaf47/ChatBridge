# ChatBridge — チャット翻訳ツール 🌉

ゲーム中のチャットをリアルタイムで翻訳するデスクトップツール。
ホットキー一発で入力中のテキストを翻訳し、そのまま送信できます。

> 🌐 [English version](README_EN.md)

## 特徴

- 🎮 **ホットキー翻訳**: `Alt+J` で選択/入力中のテキストを瞬時に翻訳
- 🌏 **多言語対応**: 日本語⇔英語をはじめ、9言語間の翻訳に対応
- 🔄 **オーバーレイ表示**: ゲーム画面の上に翻訳結果を表示（フォーカスを奪わない）
- ⚙️ **設定画面**: ホットキー、翻訳方向、UI言語などをカスタマイズ
- 🌐 **UIローカライゼーション**: 日本語/英語UI内蔵、外部JSONで言語追加可能
- 🛡️ **管理者権限対応**: ゲーム上でのキー入力シミュレーションに必要な権限を自動管理
- 🚀 **自動起動**: タスクスケジューラによる管理者権限での自動起動に対応
- 🔒 **二重起動防止**: 管理者/通常権限をまたいだ多重起動を防止
- 💰 **無料**: MyMemory翻訳API（APIキー不要）

## インストール

`ChatBridge.exe` を任意のフォルダに配置するだけです。
インストーラーは不要で、レジストリやシステムフォルダを汚しません。

```
📁 ChatBridge/
  ├── ChatBridge.exe
  └── config.json（初回起動時に自動生成）
```

## 使い方

### 初回起動

1. `ChatBridge.exe` をダブルクリック（または右クリック →「管理者として実行」）
2. 言語選択ダイアログが表示されるので、日本語/English を選択
3. 管理者権限が必要な場合、自動的に昇格ダイアログが表示されます
4. Windows 起動時の自動起動を設定するか確認されます
5. システムトレイにアイコンが表示されれば起動完了

### ⚠️ 管理者権限について

**原神、Valorant、Apex Legends** などの管理者権限で実行されるゲームで使用する場合、
ChatBridge も**管理者権限で起動する必要があります**。

通常権限で起動した場合は、自動的に管理者権限への昇格を案内します。

> **仕組み**: Windows のセキュリティ制限（UIPI）により、
> 通常権限のアプリは管理者権限アプリのキー入力を検知できません。

### 翻訳操作

1. ゲームのチャット欄にテキストを入力する
2. `Alt+J` を押す
3. オーバーレイに翻訳結果が表示される
4. `Enter` → 翻訳結果をペースト / `Esc` → キャンセル

### 設定

システムトレイのアイコンを右クリック → 「設定を開く」

| 設定項目 | 説明 |
|---|---|
| ホットキー | 翻訳を実行するキー（デフォルト: `Alt+J`） |
| 翻訳方向 | 翻訳元/翻訳先の言語ペア |
| オーバーレイ位置 | カーソル位置 / 画面中央 / 右上 |
| オーバーレイ不透明度 | 30%〜100% |
| 自動ペースト | 確認なしで即座にペースト |
| 自動起動 | Windows 起動時にタスクスケジューラで自動起動 |
| UI言語 | 日本語 / English |
| MyMemory メール | 登録すると1日50,000文字まで利用可能 |

## 自動起動の設定

1. トレイアイコン右クリック → 「設定を開く」
2. 「Windows 起動時に自動的に開く」にチェック
3. UAC プロンプトが表示されるので「はい」を選択
4. 以後、Windows 起動時に管理者権限で自動起動します（UAC は表示されません）

> **仕組み**: Windows タスクスケジューラを使用。
> タスク登録時のみ UAC の確認が必要で、以降の自動起動時には UAC は表示されません。

## アンインストール

1. ChatBridge を終了する
2. 自動起動を設定している場合は、事前に**設定画面で自動起動を OFF** にするか、
   タスクスケジューラから `ChatBridge` タスクを手動削除する
3. ChatBridge のフォルダごと削除で完了

> **ポータブル設計**: 自動起動設定（タスクスケジューラ）を除き、
> フォルダ外にファイルやレジストリを作成しません。

## UIローカライゼーション

日本語・英語はアプリに内蔵されています。
他の言語を追加するには:

1. アプリと同じフォルダに `locales/` フォルダを作成
2. `locales/ko.json` のようにJSONファイルを配置
3. 設定画面で新しい言語を選択

JSON ファイルのフォーマットは [i18n.py](i18n.py) の `_BUILTIN` 辞書を参照してください。
