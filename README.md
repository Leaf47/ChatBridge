# ChatBridge — チャット翻訳ツール 🌉

ゲーム中のチャットをリアルタイムで翻訳するデスクトップツール。
ホットキー一発で入力中のテキストを翻訳し、そのまま送信できます。

## 特徴

- 🎮 **ホットキー翻訳**: `Alt+J` で選択/入力中のテキストを瞬時に翻訳
- 🌏 **多言語対応**: 日本語⇔英語をはじめ、9言語間の翻訳に対応
- 🔄 **オーバーレイ表示**: ゲーム画面の上に翻訳結果を表示
- ⚙️ **設定画面**: ホットキー、翻訳方向、UI言語などをカスタマイズ
- 🌐 **UIローカライゼーション**: 日本語/英語UI内蔵、外部JSONで言語追加可能
- 💰 **無料**: MyMemory翻訳API（APIキー不要）

## インストール

`ChatBridge.exe` を任意のフォルダに配置するだけです。
インストーラーは不要で、レジストリやシステムフォルダを汚しません。

```
📁 ChatBridge/
  ├── ChatBridge.exe
  └── README.md
```

初回起動時に `config.json`（設定ファイル）が同じフォルダに自動生成されます。

## 使い方

### 起動

`ChatBridge.exe` をダブルクリックで起動します。
システムトレイにアイコンが表示されれば起動完了です。

### ⚠️ 管理者権限について（重要）

**原神、Valorant、Apex Legends** などの管理者権限で実行されるゲームで使用する場合、
ChatBridge も**管理者権限で起動する必要があります**。

これは Windows のセキュリティ制限（UIPI）によるもので、
通常権限のアプリは、管理者権限のアプリのキー入力を検出できません。

**手動で管理者権限で起動する方法:**
1. `ChatBridge.exe` を右クリック
2. 「管理者として実行」を選択
3. UACプロンプトで「はい」をクリック

**自動で管理者権限起動する方法:**
1. ChatBridge を起動（初回は右クリック→管理者として実行）
2. システムトレイアイコンを右クリック →「設定を開く」
3. 「Windows 起動時に自動的に開く」にチェック
4. 「管理者権限で起動する」にチェック
5. UACプロンプトが表示されるので「はい」を選択
6. 以後、Windows 起動時に自動で管理者権限起動します（UACは表示されません）

> **仕組み**: 管理者権限での自動起動は Windows タスクスケジューラを使用します。
> タスク登録時のみ UAC の確認が必要で、以降の自動起動時には UAC は表示されません。
> 通常権限での自動起動はレジストリ（`HKCU\Software\Microsoft\Windows\CurrentVersion\Run`）を使用します。

### 翻訳操作

1. 翻訳したいテキストがあるウィンドウをアクティブにする
2. `Alt+J` を押してキーを離す
3. オーバーレイに翻訳結果が表示される
4. `Enter` → 翻訳結果をペースト / `Esc` → キャンセル

### 設定

システムトレイのアイコンを右クリック → 「設定を開く」

## アンインストール

1. ChatBridge を終了する
2. 自動起動を設定している場合は、事前に**設定画面で自動起動をOFF**にするか、以下を手動で行う:
   - **通常権限の自動起動**: レジストリ `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` から `ChatBridge` を削除
   - **管理者権限の自動起動**: タスクスケジューラから `ChatBridge` タスクを削除
3. ChatBridge のフォルダごと削除で完了

> **ポータブル設計**: ChatBridge は上記の自動起動設定を除き、
> フォルダ外にファイルやレジストリを作成しません。

## UIローカライゼーション

日本語・英語はアプリに内蔵されています。
他の言語を追加するには:

1. アプリと同じフォルダに `locales/` フォルダを作成
2. `locales/ko.json` のようにJSONファイルを配置
3. 設定画面で新しい言語を選択

JSONファイルのフォーマットは [i18n.py](i18n.py) の `_BUILTIN` 辞書を参照してください。
