## [Unreleased]

> ⚠️ 以下の変更は v2.0.0 としてリリース予定。実機テスト・OCR 精度検証が未完了のため未リリース。

### 追加
- 受信翻訳: OCR 画面翻訳機能（画面キャプチャ + Tesseract OCR + 差分検出 + 翻訳パイプライン）
- 受信翻訳: エリア指定 UI（ドラッグで翻訳対象のチャット領域を選択）
- 受信翻訳: 受信翻訳オーバーレイ（時系列メッセージ表示 + ステータスバー）
- 受信翻訳: オーバーレイ自動非表示（ターゲットアプリがアクティブな時のみ表示）
- 受信翻訳: ターゲットアプリの自動検出 + 手動指定

### 修正
- 受信翻訳: `recv_source_lang` / `recv_target_lang` が `null` のとき翻訳が失敗するバグを修正

### 改善
- 受信翻訳: オーバーレイにステータスバーを追加（パイプラインの動作状態をリアルタイム表示）
- 受信翻訳: エラー発生時のユーザーへの通知を改善（連続失敗時のダイアログ表示等）

### ドキュメント
- `docs/` フォルダを新設し、開発者向けドキュメントを整理
- ARCHITECTURE.md, SPECIFICATION.md, ROADMAP.md, TROUBLESHOOTING.md, DEVELOPMENT_LOG.md, TASKS.md を追加・更新
- ドキュメント命名規則を統一（略語廃止: SPEC → SPECIFICATION）

### 追加
- 自動アップデートチェック機能（GitHub Releases 経由）

## [1.0.0] - 2026-04-10

### 追加
- ホットキー（Alt+J）による送信翻訳
- MyMemory API 翻訳エンジン
- フローティングオーバーレイ（Enter確定 / Escキャンセル）
- システムトレイ常駐
- 管理者権限での自動再起動
- Windows 起動時の自動起動（タスクスケジューラ）
- 設定画面（一般/翻訳エンジン/About タブ）
